## 一、这玩意儿是干啥的？

简单说，**DeepReader 就是个"网页内容抓取器"**。

你给它一个网址，它自动把网页里的文字抠出来，整理得干干净净，保存到你的手机里。

### 它能抓取啥？

| 网站 | 能抓啥 |
|------|--------|
| 🚀 **零配置** | 无需 API Key，无需登录 |
| 🌐 **多平台支持** | Twitter/X、Reddit、YouTube、任意网页 |
| 📝 **Markdown 输出** | 自动清洗为干净格式 |
| 💾 **自动保存** | 保存到智能体记忆库 |
| 🔍 **智能路由** | 自动识别 URL 类型并选择解析器 |
| 🎙️ NotebookLM 集成| 支持将内容上传到 Google NotebookLM 并生成多种格式

### 有啥用？

- 看到好文章，抓下来慢慢看
- 推特长文，保存下来做笔记
- YouTube 视频，提取字幕做总结
- 每天收集行业资讯，自动整理

## 二、安装（3 分钟搞定）

### 方法 1：一键安装（最简单）

打开电脑的命令行（黑窗口），输入：

```
npx clawhub@latest install deepreader
```

等它自己跑完，显示"成功"就装好了。

### 方法 2：如果上面不行（网络问题）

**第一步**：配置代理（让电脑能访问国外网站）

```
设置 → 网络 → 代理 → 开启代理
```

或者在命令行输入：

```
set HTTP_PROXY=http://127.0.0.1:10808
set HTTPS_PROXY=http://127.0.0.1:10808
```

**第二步**：下载安装

```
cd C:\Users\Administrator\.openclaw\skills
git clone https://github.com/astonysh/OpenClaw-DeepReeder.git
```

**第三步**：装依赖

```
cd OpenClaw-DeepReeder
pip install -e .
```

搞定！

## 三、怎么用？（超简单）

### 方法 1：直接发消息（最常用）

**打开你的 OpenClaw 聊天窗口，直接发网址就行！**

比如：

```
帮我看看这个：https://x.com/elonmusk/status/123456
```

```
读一下这篇文章：https://example.com/blog/ai-trends
```

```
把这个视频的字幕提出来：https://youtube.com/watch?v=xxx
```

就这么简单！DeepReader 会自动抓取，整理好，保存起来。

### 方法 2：批量处理（一次抓多个）

如果你想一次抓取好几个网址，可以这样说：

```
帮我读一下这几个链接：
1. https://x.com/user1/status/111
2. https://x.com/user2/status/222
3. https://example.com/article
```

它会自动一个接一个处理，最后告诉你都保存好了。

## 四、高级玩法：生成学习资料

> 这部分需要你有 Google 账号，并且能访问 Google 服务

### 啥是 NotebookLM？

Google 出的一个学习工具，能把文章变成：
- 🎙️ **音频**（像播客一样，可以听）
- 🧠 **思维导图**（知识结构图）
- 📇 **抽认卡**（复习用的小卡片）
- ❓ **测验题**（测试你学会了没）

### 怎么用？

#### 1. 先认证（只用做一次）

在命令行输入：

```
notebooklm login
```

它会给你一个链接，用浏览器打开，用 Google 账号登录，授权一下，就搞定了。

#### 2. 生成音频（听文章）

```
帮我把这篇文章变成音频，我想通勤路上听
https://example.com/article --audio
```

它会生成一个 MP3 文件，大概 10-15 分钟，两个人像聊天一样把文章内容讲出来。

#### 3. 生成思维导图（理思路）

```
帮我画个思维导图，让我快速理解这篇文章
https://example.com/article --mindmap
```

它会生成一张图，中心是主题，分支是各个知识点，一目了然。

#### 4. 生成抽认卡（复习用）

```
帮我生成抽认卡，我要复习这篇文章的内容
https://example.com/article --flashcards
```

它会生成一堆小卡片，正面是问题，背面是答案，你可以用来复习。

#### 5. 全套都来一遍（懒人必备）

```
帮我把这篇文章做成完整的学习资料包
https://example.com/article --all
```

它会生成：
- ✅ 原文（Markdown 格式）
- ✅ 音频（MP3）
- ✅ 思维导图（图片）
- ✅ 抽认卡（可以导入 Anki）
- ✅ 测验题（自测用）

## 五、实际使用场景

**问题**: 每天要看很多推文、文章，记不住怎么办？

它会提取全部字幕，你可以搜索关键词，快速定位。

```

https://example.com/topic1
https://example.com/topic2
https://example.com/topic3
--flashcards --quiz
```

它会生成抽认卡和测验题，帮你复习。

**问题**: 要看很多论文，记不住要点？

**解决**:

```

https://example.com/paper1
https://example.com/paper2
https://example.com/paper3
--summary
```

它会给每篇论文生成摘要，方便你快速浏览。

## 六、常见问题（一看就懂）

### Q1: 装的时候说"Rate limit exceeded"是啥意思？

**答**: GitHub 限制你了，等几分钟再试就行。

### Q2: 访问不了 GitHub 怎么办？

**答**: 开代理！开代理！开代理！（重要的事情说三遍）

### Q3: 抓取推特失败？

**答**: 
1. 检查代理开了没
2. 推特有时候抽风，等会儿再试
3. 实在不行，用网页版推特复制链接

### Q4: YouTube 字幕提不出来？

**答**: 那个视频可能没有字幕。找个有字幕的视频试试。

### Q5: 抓下来的内容是乱码？

**答**: 命令行编码问题。试试：

```
chcp 65001
```

然后再运行。

### Q6: 保存的文件在哪？

**答**: 一般在：

```
C:\Users\Administrator\.openclaw\skills\memory\inbox\
```

打开这个文件夹，就能看到抓下来的文章。

### Q7: 能不能抓取付费文章？

**答**: 不能。人家收费的，咱不能白嫖。

## 七、注意事项（必看！）

### ✅ 推荐做的事

1. **定期清理**: 抓下来的文件会越来越多，定期删掉不用的
2. **起个好名字**: 网址尽量有意义，方便以后找
3. **一次别太多**: 一次抓取别超过 20 个网址，会卡
4. **重要内容备份**: 抓下来的重要文章，另存一份

### ❌ 别做的事

1. **别抓付费内容**: 违法的
2. **别狂抓同一个网站**: 人家会封你
3. **别全信**: 抓下来的内容，自己判断真假
4. **别乱分享**: 有些内容有版权
