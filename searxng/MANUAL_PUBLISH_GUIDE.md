## 📋 发布步骤

### 1. 打开 HTML 文件

使用浏览器打开：
```
C:\Users\Administrator\.openclaw\workspace\searxng-article-wechat.html
```

或者双击文件直接在默认浏览器中打开。

### 2. 复制内容

在浏览器中：
- 按 `Ctrl + A` 全选
- 按 `Ctrl + C` 复制

### 3. 登录微信公众平台

访问：https://mp.weixin.qq.com

使用管理员微信扫码登录。

### 4. 创建新草稿

左侧菜单：
- **草稿箱** → **新的创作**

### 5. 粘贴内容

在编辑器中：
- 按 `Ctrl + V` 粘贴

格式会自动保留，包括：
- ✅ 标题样式
- ✅ 代码块高亮
- ✅ 表格
- ✅ 引用块
- ✅ 列表

### 6. 填写元数据

**标题**（必填）：
```
隐私搜索神器！本地部署 SearXNG，彻底摆脱大数据追踪
```

**作者**（可选）：
```
YanG
```

**摘要**（可选，建议填写）：
```
不用 Google、不用百度，这款开源元搜索引擎聚合了 242+ 搜索服务，不追踪、不画像、不记录，完全掌控你的搜索隐私。10 分钟部署，终身隐私保护。
```

**封面图**（必填）：
1. 点击"封面图"区域
2. 选择"从本地上传"
3. 上传文件：`C:\Users\Administrator\.openclaw\skills\wechat-draft-publisher\cover.png`
4. 裁剪并确认

### 7. 预览（可选）

点击"预览"：
- 发送到手机上预览效果
- 检查格式是否正确
- 确认图片显示正常

### 8. 保存/发表

**保存到草稿箱**：
- 点击"保存"
- 文章保存在草稿箱，稍后可以继续编辑或定时发表

**直接发表**：
- 点击"发表"
- 文章立即发布到公众号
