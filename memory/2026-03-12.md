# 2026-03-12 - Mazda KPI Report Fixes

## CSI 报表修复

### 问题 1: 评价工单结算范围显示错误
- **现象**: "评价工单结算范围"显示的是大区值（如"西区"），而不是时间范围
- **原因**: CSI 统计表列头格式为"字段名：时间范围"，行内数据第一列是经销商代码，第二列是经销商名称，后续列包含大区值
- **修复**: 从列头解析出时间范围（如"2 月 1 日 -3 月 8 日"），而不是取行内大区值

### 新增指标
1. **直评日期范围**: 从统计表头"直评日期：xxxx-xxxx"解析
2. **本月维修合同数**: 从统计表头"本月维修合同：xxxx-xxxx"解析，带时间范围后缀

### 问题 2: HTML 报表未渲染
- **现象**: HTML 文件打开显示的是 Markdown 源码，没有表格渲染
- **原因**: 之前用`<pre>{md_text}</pre>`直接包裹 Markdown 原文
- **修复**: 改为结构化 HTML 渲染，使用真正的`<table>`、`<tr>`、`<td>`标签，添加 CSS 样式（卡片布局、蓝色标题、表格边框）

## 文件修改
