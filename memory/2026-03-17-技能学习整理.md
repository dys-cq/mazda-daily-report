# OpenClaw 技能学习整理

## 📦 已安装技能总览

我目前能用的技能一共有 **41 个**，分几个大类：

### 不需要 API Key 的技能（开箱即用）

这些技能装好就能用，不用配置任何东西：

1. **coding-agent** - 调用 Codex/Claude Code 写代码
2. **healthcheck** - 主机安全检查和加固
3. **node-connect** - 诊断 OpenClaw 节点连接问题
4. **openai-whisper** - 本地语音转文字（用 Whisper 模型）
5. **skill-creator** - 创建或更新技能包
6. **video-frames** - 用 ffmpeg 提取视频帧
7. **weather** - 查天气（wttr.in 或 Open-Meteo，免费）
8. **baidu-scholar-search-skill** - 搜中英文学术文献
9. **baoyu-compress-image** - 压缩图片转 WebP
10. **baoyu-danger-x-to-markdown** - 把 X/Twitter 推文转 markdown
11. **baoyu-format-markdown** - 格式化 markdown 文档
12. **baoyu-markdown-to-html** - Markdown 转 HTML（微信风格）
13. **baoyu-url-to-markdown** - 网页转 markdown
14. **mazda-daily-report** - 马自达售后日报生成
15. **multi-search-engine** - 多搜索引擎搜索（17 个引擎）

## 📋 飞书多维表格说明

**注意**：飞书技能目前支持读写飞书文档（docx）、云盘、权限、知识库，但**不直接支持创建多维表格（bitable）**。

有两种方案：

### 方案一：用飞书文档创建表格（推荐）
- 我可以用 `feishu-doc` 创建一个飞书文档，里面用表格形式管理技能
- 优点：立即可用，不需要额外权限
- 缺点：不是真正的多维表格，功能有限

### 方案二：手动创建多维表格
- 你在飞书里手动创建一个多维表格
- 把表格分享给我（机器人）
- 我用飞书 API 帮你填充数据
- 优点：功能强大，可筛选排序
- 缺点：需要你手动创建第一步
