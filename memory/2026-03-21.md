### 核心实现方案（用户明确指定，无需 API Key）

**抖音视频下载：**
- 前端提供 JS 代码，用户复制后在抖音网页 Console 执行
- JS 调用抖音内部 API: `https://www.douyin.com/aweme/v1/web/aweme/detail/?aweme_id={视频 ID}&aid=6383&cookie_enabled=true`
- 用户把获取到的直链粘贴回应用
- 后端用 Python urllib 下载视频（携带 User-Agent 和 Referer 请求头）
- **不需要任何 API Key**

**B 站视频下载：**
- 使用 yt-dlp 命令行工具下载
- 支持 `--cookies-from-browser chrome` 读取登录状态
- **不需要任何 API Key**

**音频处理：**
- FFmpeg 提取 MP3: `ffmpeg -i video.mp4 -vn -acodec libmp3lame -q:a 0 audio.mp3`
- FFmpeg 调整音量：`ffmpeg -i audio.mp3 -af "volume={dB}dB" -y output.mp3`
- 音量范围：-20dB 到 +10dB
