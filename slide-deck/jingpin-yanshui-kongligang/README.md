# 竞聘演讲 PPT - 孔立刚

## 项目说明

本项目使用 baoyu-slide-deck 技能规范，为孔立刚先生生成了专业的竞聘演讲 PowerPoint 幻灯片。

## 生成内容

### PPT 文件
- **文件名**: 竞聘演讲 - 孔立刚.pptx
- **幻灯片数量**: 17 页
- **尺寸**: 16:9 宽屏
- **风格**: 专业企业风格（海军蓝 + 白色 + 金色）

1. **封面页** - 竞聘演讲 / 演讲人：孔立刚

4. **个人劣势分析** - 3 个待提升领域

6. **自动化工具应用** - 4 项自动化成果

8. **核心工作思路** - 三大核心原则（数据驱动、客户至上、团队赋能）
9. **业务规划第一阶段** - 夯实基础 (1-3 个月)
10. **业务规划第二阶段** - 优化提升 (4-9 个月)
11. **业务规划第三阶段** - 创新突破 (10-18 个月)
12. **团队管理设想 1** - 建立清晰的目标体系
13. **团队管理设想 2** - 打造学习型组织
14. **团队管理设想 3** - 强化沟通与协作

## 设计规范

### 配色方案
- **海军蓝** (#1a365d) - 主色调，用于标题栏和强调
- **白色** (#ffffff) - 背景色
- **金色** (#d69e2e) - 强调色，用于副标题和重点
- **浅灰色** (#f7fafc) - 辅助背景

### 字体
- **中文字体**: Microsoft YaHei (微软雅黑)
- **标题**: 36-54pt，加粗
- **正文**: 18-24pt

### 布局特点
- 专业的企业风格
- 清晰的视觉层次
- 每页 2-4 个关键点
- 16:9 宽屏比例，适合现代演示设备

## 使用方法

### 查看 PPT
直接打开 `竞聘演讲 - 孔立刚.pptx` 文件即可查看演示文稿。

### 修改内容
如需修改内容，可以：
1. 直接编辑 PPT 文件
2. 或修改 `generate_ppt.py` 脚本后重新运行

### 重新生成 PPT
```bash
cd slide-deck/jingpin-yanshui-kongligang
uv run python generate_ppt.py
```

## 技术说明

- **生成工具**: Python + python-pptx 库
- **Python 版本**: 3.13.5
- **包管理**: uv
- **依赖**: python-pptx

## 备注

由于 bun 运行时环境安装失败，本项目采用 Python 脚本直接生成 PPT 文件的方式，而非使用 baoyu-slide-deck 的图像生成功能。最终生成的 PPT 文件为纯文本和矢量图形，文件体积小，兼容性好，可直接用于演示。

如需生成带有 AI 生成图像的增强版幻灯片，需要先解决 bun 运行时环境的安装问题，然后使用 `batch.json` 文件调用 baoyu-image-gen 技能生成幻灯片图像。
