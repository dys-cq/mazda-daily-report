
3. **按照提示完成配置**：
   - 选择图像生成提供商 (OpenAI/Google/DashScope/Replicate)
   - 配置 API 密钥
   - 确认生成参数

4. **等待生成完成**：
   - 系统会自动生成 15 张幻灯片图片
   - 自动生成 PPTX 和 PDF 文件
   - 输出位置：`slide-deck/mazda-skyactiv-tech/`

### 方法二：手动配置后使用图像生成

1. **配置图像生成偏好**：
   创建配置文件 `~/.baoyu-skills/baoyu-image-gen/EXTEND.md`：

```markdown
# Baoyu Image Gen Configuration

## Default Provider
provider: openai  # 或 google, dashscope, replicate

## Default Quality
quality: 1k

## Default Aspect Ratio
aspect_ratio: 16:9

## Default Model
default_model:
  openai: dall-e-3
  google: gemini-3-pro-image-preview
  dashscope: wanx2.1
```

2. **设置 API 密钥** (根据选择的提供商)：
   - OpenAI: `OPENAI_API_KEY`
   - Google: `GOOGLE_API_KEY`
   - DashScope: `DASHSCOPE_API_KEY`
   - Replicate: `REPLICATE_API_KEY`

3. **运行批量生成**：
```bash
cd C:\Users\Administrator\.openclaw\workspace\slide-deck\mazda-skyactiv-tech
npx -y bun "~/.openclaw/skills/baoyu-image-gen/scripts/main.ts" --promptfiles prompts/*.md --image slide.png --ar 16:9
```
